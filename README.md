# smth-android-client
水木社区的安卓客户端

项目目的以学习为主，希望以此项目为基础，学习以下内容：</br>
1 RecyclerView + CardView</br>
2 底部菜单</br>
3 MVP模式及自动化测试</br>
4 Retrofit2 + OkHttp3 + RxJAVA</br>
5 JAVA Html解析</br>
6 数据库模型</br>

# 当前支持功能：
1 热点话题列表</br>
2 板块列表</br>
3 用户登录</br>
4 Post查看</br>
